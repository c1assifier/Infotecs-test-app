# Infotecs Test App

Тестовое задание для стажера **React Frontend Developer**.

Проект представляет собой приложение базы пользователей с авторизацией, списком пользователей и возможностью создания/редактирования пользователя.

---

## Stack

- TypeScript
- React 16
- Webpack + NPM
- Ant Design 5
- styled-components
- React Router v6
- TanStack Query v4
- Axios
- dayjs
- Feature-Sliced Design (FSD)

---

## Functionality

### Authorization

- Страница логина (`/login`)
- Логин: `admin`
- Пароль: `admin`
- Имитация авторизации с задержкой 2000 мс
- Сохранение токена в `localStorage`
- Редирект авторизованного пользователя на страницу пользователей

### Users

- Страница списка пользователей (`/users`)
- Получение данных с помощью MockAPI
- Форматирование даты регистрации (`DD.MM.YYYY`)
- Кнопка выхода из системы
- Создание пользователя (modal)
- Редактирование пользователя по клику на имя или аватар

### Modals

- Создание и редактирование пользователя
- Валидация полей
- Отключение кнопок и закрытия модального окна во время запроса
- Обновление списка пользователей после успешного запроса

### 404

- Страница 404 с использованием компонента `Result` из Ant Design

---

## API

Для имитации backend используется **mockapi.io**.

Ресурс:

- `users`

Методы:

- `GET /users`
- `GET /users/:id`
- `POST /users`
- `PUT /users/:id`
- `DELETE /users/:id`
